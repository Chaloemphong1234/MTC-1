สร้างเว็บไซต์ Portfolio ส่วนตัวด้วย HTML, CSS และ JavaScript โดยมีรายละเอียดดังนี้:

1. โครงสร้างเว็บไซต์
- หน้า Home (แนะนำตัว)
- หน้า About (ประวัติส่วนตัว)
- หน้า Portfolio (ผลงาน)
- หน้า Gallery (รวมภาพหลายภาพ)
- หน้า Contact (ข้อมูลติดต่อ)

2. ดีไซน์
- ใช้โทนสีสุภาพ ดูทางการ (เช่น น้ำเงิน ขาว เทา)
- มี Navbar ด้านบนแบบติด (sticky)
- ใช้ปุ่ม (button) ที่มี animation hover
- มีการ์ด (card) สำหรับแสดงผลงาน
- ใช้ grid หรือ flexbox จัด layout

3. CSS ลูกเล่น
- Hover effect ปุ่ม และรูปภาพ
- Smooth scroll
- Animation fade-in ตอนเลื่อนหน้า
- Shadow และ border-radius ให้ดู modern

4. หน้า Home
- มีรูปโปรไฟล์
- ชื่อ-นามสกุล
- คำแนะนำตัวสั้น ๆ

5. หน้า About
- ประวัติส่วนตัว
- การศึกษา
- ทักษะ (Skill bar)

6. หน้า Portfolio
- แสดงผลงานเป็น card
- แต่ละ card มี รูป + ชื่อผลงาน + คำอธิบาย

7. หน้า Gallery
- สามารถใส่รูปภาพได้หลายรูป (อย่างน้อย 6-12 รูป)
- มี effect zoom เมื่อ hover

8. หน้า Contact
- อีเมล
- เบอร์โทร
- ลิงก์โซเชียล

9. Responsive
- รองรับมือถือ แท็บเล็ต และคอมพิวเตอร์

10. โครงสร้างไฟล์
- index.html
- style.css
- script.js
- folder images/

11. โค้ดต้องเขียนให้พร้อมใช้งานทันที และมี comment อธิบาย

ให้เขียนโค้ดครบทุกไฟล์ และแยกไฟล์ชัดเจน
